# How to contribute

## With data
This is a one-direction data dump coming from [Consurf.net](https://consurf.net). All moderation and validation are done on the website, which gets exported to this repository on a regular basis.

Consurf is setup in a way you can login with your Bluesky account and start contributing.

## With support
Consurf is mainly run and maintained by [Nuki](https://en.wikifur.com/wiki/Nuki), you can support this project with donations. Consurf runs in a VPS and has costs currently covered out-of-pocket by him, any contribution is greatly appreciated!

The current estimation is about `$22 CAD` per month of costs. All supporters will be recognized (opt out available) in our supporter page once we start having.
